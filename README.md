# بوت تليجرام لتفعيل الأرقام (SMS-Man API)

هذا البوت يتيح للمستخدمين شراء أرقام افتراضية من جميع أنحاء العالم وتفعيل مختلف الخدمات (مثل واتساب، تلجرام، إلخ) باستخدام واجهة برمجة تطبيقات SMS-Man.com.

## المميزات:
- دعم جميع الدول والخدمات المتاحة على SMS-Man.
- نظام بحث ذكي عن الدول والخدمات.
- واجهة مستخدم احترافية باستخدام الأزرار الشفافة (Inline Buttons).
- تحديث فوري للرصيد.
- إدارة كاملة للعمليات (طلب، جلب كود، إلغاء).

## متطلبات التشغيل:
1. توكن البوت من @BotFather.
2. مفتاح API من موقع SMS-Man.com.

## النشر على Railway:
1. ارفع الملفات إلى مستودع GitHub الخاص بك.
2. اربط المستودع بمنصة Railway.
3. أضف متغيرات البيئة (Variables):
   - `BOT_TOKEN`: توكن البوت الخاص بك.
   - `SMS_API_KEY`: مفتاح API الخاص بـ SMS-Man.
4. سيقوم Railway بتشغيل البوت تلقائياً باستخدام ملف `Procfile`.
